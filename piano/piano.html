Basic mode: press and hold "Shift" and press num button for make offset

Advanced mode: press and hold "Shift". Pressing num button will change active layout. Setting "Edit" checkbox to true will toggle editing mode. While "Edit" is true pressing and releasing one of displaying keys will select octave and pressing num button then will set selected octave to corresponding offset.

Pro mode: press and hold "Shift". Pressing num button will change active layout. For assigning: enter node code to input field and press button. Then set "Edit" checkbox to true and press and hold "Shift". Pressing one of displaying keys will select key for note code.
